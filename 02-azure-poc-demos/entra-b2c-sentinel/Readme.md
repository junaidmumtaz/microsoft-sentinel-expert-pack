## Overview

This proof of concept (POC) demonstrates how to successfully onboard **Microsoft Entra B2C logs** from multiple tenants (Tenant A and Tenant B) into **Microsoft Sentinel**.  

The blog provides a step-by-step walkthrough covering:
- Configuration steps in Entra B2C
- Diagnostic settings and data export
- Sentinel data connector setup
- Verification and validation of ingested logs
- Lessons learned and success outcomes
